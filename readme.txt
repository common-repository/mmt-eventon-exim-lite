Show Eventbrite events on website and create Eventbrite events directly from Wordpress.


== Description ==

This plugin connects with Eventbrite and allows users to show events on Wordpress site and also create Eventbrite events 
from Wordpress site itself.

Documentation: <a href="http://momothemes.com/documentationEXIM" target="_blank">Read documentation</a>

**Display Eventbrite Events**

Events will be pulled from Eventbrite and a custom widget display them as list. This automated and only needs adding eventbrite API and enabling it in the settings panel.

**Export Events to Eventbrite**

For EventON users, it allows exporting posts as events to Evenbrite automatically.

== Installation ==

= This plugin can be installed directly from your site. =

1. Log in and navigate to Plugins & Add New.
2. Type "exim" into the Search input and click the "Search" button.
3. Locate the "exim" in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.

= It can also be installed manually. =

1. Download the "mmt-eventon-exim-lite.zip" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "exim" in the list and click the "Activate" link.
